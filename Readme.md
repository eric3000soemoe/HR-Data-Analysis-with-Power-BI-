HR Analytics Dashboard - Headcount & Employee Turnover


📌 Project Overview
This project presents a comprehensive Power BI Dashboard designed to analyze company Headcount and Employee Turnover trends. It is built to assist HR departments in understanding workforce dynamics and making data-driven decisions.

📊 Key Insights & Features
1. Headcount Report
Total Headcount: Displays a total workforce of 3,409 employees.

Demographics: Provides a breakdown by age, gender (61% Male, 39% Female), and education level.

Departmental Distribution: Highlights that the Software and Sales departments have the highest headcount.

Work Location: Reveals that 80% of the workforce operates remotely, while 20% work on-site.

2. Employee Turnover Report
Turnover Rate: Shows a total turnover rate of 43%.

Termination Type: Indicates that the majority of departures are voluntary.

Turnover by Year: Tracks trends from 2012, identifying a peak in 2014 and a significant decline to 0.15% by 2022.

Top Reasons for Leaving: Identifies "Found a better opportunity" and "More flexible benefits" as the primary drivers for turnover.

🛠️ Tools Used
Power BI Desktop: Used for data visualization and dashboard construction.

Power Query: Utilized for data transformation and cleaning.

DAX (Data Analysis Expressions): Employed for calculating key metrics and measures.

📂 Repository Contents
HR_Analytics_Report.pbix: The source Power BI project file.

Data_Source/: Contains the sample dataset (Excel/CSV) used for the dashboard.

🚀 How to View
Download the .pbix file and open it using Power BI Desktop.

Interact with the dashboard by utilizing the year filters and department slicers to explore the data dynamically.

HR Analytics Dashboard - Headcount & Employee Turnover
📌 Project Overview
ဒီ Project ကတော့ ကုမ္ပဏီတစ်ခုရဲ့ ဝန်ထမ်းအင်အားစု (Headcount) နဲ့ ဝန်ထမ်းနှုတ်ထွက်နှုန်း (Turnover) တို့ကို အသေးစိတ် ခွဲခြမ်းစိတ်ဖြာထားတဲ့ Power BI Dashboard ဖြစ်ပါတယ်။ 
HR ဌာနအနေနဲ့ ဝန်ထမ်းတွေရဲ့ အခြေအနေကို ပိုမိုနားလည်စေရန်နဲ့ Data-driven decisions များ ချမှတ်နိုင်ရန် ရည်ရွယ် တည်ဆောက်ထားတာ ဖြစ်ပါတယ်။

📊 Key Insights & Features
1. Headcount Report (ဝန်ထမ်းအင်အားစု)
Total Headcount: စုစုပေါင်း ဝန်ထမ်း ၃,၄၀၉ ဦး ရှိကြောင်း ဖော်ပြထားပါတယ်။

Demographics: အသက်အရွယ်၊ ကျား/မ ခွဲခြားမှု (Male 61%, Female 39%) နဲ့ ပညာအရည်အချင်းအလိုက် အသေးစိတ် ကြည့်ရှုနိုင်ပါတယ်။

Department-wise Distribution: Software နဲ့ Sales ဌာနတွေမှာ ဝန်ထမ်းအများဆုံး ရှိနေတာကို တွေ့ရပါတယ်။

Work Location: ဝန်ထမ်း ၈၀% ဟာ Remote (အဝေးရောက်) လုပ်ကိုင်နေကြပြီး ၂၀% သာ On-site ဆင်းကြပါတယ်။

2. Employee Turnover Report (ဝန်ထမ်းနှုတ်ထွက်မှု)
Turnover Rate: စုစုပေါင်း Turnover rate ၄၃% ရှိပါတယ်။

Termination Type: ဝန်ထမ်းအများစုဟာ ကိုယ့်သဘောဆန္ဒအလျောက် (Voluntary) နှုတ်ထွက်ကြတာ များပါတယ်။

Turnover by Year: ၂၀၁၄ ခုနှစ်မှာ နှုတ်ထွက်နှုန်း အမြင့်ဆုံးဖြစ်ခဲ့ပြီး ၂၀၂၂ မှာတော့ ၀.၁၅% အထိ သိသိသာသာ လျော့ကျသွားတာကို တွေ့ရပါတယ်။

Top Reasons for Leaving: "Found a better opportunity" နဲ့ "More flexible benefits" တို့ဟာ နှုတ်ထွက်ရခြင်းရဲ့ အဓိက အကြောင်းရင်းတွေ ဖြစ်ပါတယ်။

🛠️ Tools Used
Power BI Desktop: Data Visualization နဲ့ Dashboard တည်ဆောက်ရန်။

Power Query: Data Transformation နဲ့ Cleaning လုပ်ရန်။

DAX (Data Analysis Expressions): လိုအပ်တဲ့ Metrics (Calculated Measures) တွေ တွက်ချက်ရန်။

📂 Files in this Repository

၂။ Dashboard ရဲ့ Interaction တွေကို စမ်းသပ်ကြည့်ဖို့ Year Filter တွေနဲ့ Department Slicers တွေကို အသုံးပြုနိုင်ပါတယ်။
